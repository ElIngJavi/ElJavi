MariHe App — Sistema de Conteo de Productos

React 18 · FastAPI · MongoDB · Python · Vercel

Sobre el proyecto

Aplicación web fullstack para el conteo y registro de productos en tiempo real. Desarrollada para una empresa local en Tijuana, incluye autenticación JWT, roles de usuario, exportación a Excel y panel de administrador completo.

React 18FastAPIMongoDB PythonVercelJWT

Características principales

  • Autenticación con JWT y roles de usuario (admin, operador).
  • Conteo de productos en tiempo real con actualización automática.
  • Exportación de reportes a Excel con filtros por fecha.
  • Panel de administrador para gestión de usuarios y productos.
  • Diseño responsive optimizado para tablets y móviles.
